My name is Catherine, but I go by Catrina and I am an Earth Systems Researcher. Currently I am and ORISE Postdoc with the US. EPA working on the National Aquatic Resources Survey to look at nitrogen processing in the nations wetlands. I have a Ph.D. in Oceanography from the University of Rhode Island, and a Bachelors in Environmental Engineering from the University of Connecticut. I have a wide range of interests and experience, and my goal is to build a portfolio using this website. Topics I plan to highlight are data visualization, science and art collaborations, as well as the ocean and making large scale ecosystem connections.


I am building this website from the bottom up in HTML and javaScript using GitHub Pages. To keep the design modular, I wrote a simple complier using the package "htmltools" in R! Check out these links to see the code behind the scenes in my GitHub repositories. Currently many of the pages still need spell checking and I am learning how to format things for multiple screens. One of my goals is to learn how to use D3 on the web, most of my prior visualizations have been made in MatLab, R, and Python. Currently working on locally building the page: open bash from the folder with my website, run py -m http.server go to http://localhost:8000/index.html
