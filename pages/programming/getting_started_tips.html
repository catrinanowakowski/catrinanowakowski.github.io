Some tips for getting started with your website

Sizing things

I first started trying to scale everything DIY, that is hard, and while it was good for me getting to know HTML syntax, it is not what I currently do now. Bootstrap does a lot of the work for you, and it auto accounts for different screen types and sizes as well.

Rendering pages locally

Rendering pages locally before pushing them to github saves a lot of time. For my work flow, I am using the sublime text editor and pushing my files to github using "Github Desktop". To locally render a page on my website, I navigate to the folder I have the index.html file saved to, right click, and open "Git Bash Here". From there you I python to launch a local server: "py -m http.server". Now I open up an Internet browser and navigate to "http://localhost:8000/index.html". If you want to load a file with in a subfolder it would look like this: "http://localhost:8000/pages/projects/projects_main.html" assuming the "pages" folder is in the same location as your index.html file.
